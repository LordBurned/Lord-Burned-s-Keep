What is this about

This website is for organizing and showcasing the game I’m creating while keeping myself motivated. It’s a space for me to write extensively about my ideas, share how I want to build new connections on the internet, and meet new people. This is my first game project, I'm not a amazing artist or programmer but everyone needs to start from somewhere. And this website I'll post every progress and make available for anyone who wants.

Who am I

About the game

My main idea is a cyberpunk with fantasy, the humanity sprawling trought the universe. It will be a single player RPG and some strategic elements. The inspirations are Warhammer, Rimworld and some others.
